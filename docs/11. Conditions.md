# Conditions

## If, Else

```go
x := 3

if x == 1 {
    println("One")
} else if x == 2 {
    println("Two")
} else if x == 3 {
    println("Three")
} else {
    println("Out of range")
}
```

## Switch, Case

```go
x := 3

switch x {
case 1:
    println("One")
case 2:
    println("Two")
case 3:
    println("Three")
default:
    println("Out of range")
}
```

> Result

```sh
Three
```
